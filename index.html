WELCOME TO RESEARCH POINT

Research point is an application that can be used by students and researchers. The application will be built using cloud technology (Amazon Web Services). Using the various services of AWS, the flow will be well authenticated and secure. The application aims to be a point of interaction among the people who are researching on a particular topic (helping for the literature survey). This platform can also be useful for people who want to learn about the emerging technologies and trends.
